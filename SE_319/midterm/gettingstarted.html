Getting started with CSS

How to start styling your webpage

Inline vs. External vs. Internal CSS

There are three ways to implement CSS into your webpage: Inline, External, and Internal. It is important to choose the type that will suit your webpage the best. If you are only planning on changing the style of very few elements, inline CSS might be the right choice. If you want to change the style of multiple elements on your page, internal CSS can be an appropriate option. If you want to have a consistent style throughout multiple webpages, internal will be a solid option.

Inline CSS

Add the style tag to an HTML element to start modifying that element. All of your modifiers must be within double quotes

External CSS

Create a new file with the extension ".css". To change the style of the element, start by typing the name of the tag, id, or class followed by brackets. All of the CSS modifiers for that specific element or group of elements must be on separate lines within the brackets

Internal CSS

Internal CSS is very similar to external css, with the difference being that all of the css is written within the html file of the webpage.
